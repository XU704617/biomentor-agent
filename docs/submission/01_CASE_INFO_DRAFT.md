## 解决的教学问题

1. **知识点分散**：生物制造/生命科学课程横跨分子生物学、细胞生物学、代谢工程、发酵工程等多个领域，学生难以建立系统性知识网络。
2. **案例理解困难**：产业案例涉及复杂的工程化背景，学生缺乏直观的结构化分析入口。
3. **科研训练缺少结构化引导**：从知识学习到独立开展科研任务之间存在断档，缺少循序渐进的训练体系。
4. **学生难以从知识学习过渡到科研任务**：论文阅读、实验设计、学术答辩等科研核心能力训练缺乏系统性工具支撑。

## 特色与创新

### 学习闭环设计

平台围绕以下七个模块构建从知识学习到科研能力的闭环：

```text
知识探索 → 知识图谱 → 生物工具箱 → 产业案例 → 科研实战 → 学术研讨 → 学习评估
```

1. **知识探索**：基于 RAG 的领域知识检索与 AI 问答，支持多课程、多知识点的智能检索。
2. **知识图谱**：12 学科生命科学知识星图，渐进展开式 BioMind Map，可视化呈现知识关联。
3. **生物工具箱**：蛋白结构 3D 查看、质粒图谱解读、序列分析、代谢通路浏览，每个工具均集成 AI 解释。
4. **产业案例**：生物制造案例库，结构化展示技术路线与研究要点，作为科研实战的任务输入源。
5. **科研实战**：AI 驱动的科研任务生成（案例驱动 / 独立选题两种模式），结构化分解研究步骤。
6. **学术研讨**：模拟学术答辩训练器，支持材料导入、Defense Brief 生成、多轮答辩与六维评分报告。
7. **学习评估**：智能测评、学习诊断、错题记录、学习轨迹可视化。

### 技术创新点

- **AI 辅助开发**：项目核心代码由 AI 辅助生成和迭代，提示词工程驱动开发过程，可作为 AI 赋能软件开发的教学案例。
- **多模态知识呈现**：文本知识 + 3D 蛋白结构 + 质粒图谱 + 代谢通路图 + 知识星图，多维度呈现生物制造知识。
- **科研任务结构化生成**：将科研过程拆解为文献调研、实验设计、数据分析、结果讨论等结构化步骤，由 LLM 生成具体任务指导。
- **答辩模拟闭环**：从资料包导入 → 生成答辩策略 → 多轮追问 → 六维评分报告，形成完整的学术答辩训练体验。
- **国产 LLM 兼容**：支持通过 OpenAI 兼容接口接入国产大模型，不依赖特定国外 API，适合教学场景部署。
