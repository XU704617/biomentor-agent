# 页面清单

## 教师端

### 1. 教师首页 Dashboard

模块：

- 课程概览
- 已上传资料数
- 已生成题目数
- 待审核题目
- 最近测验
- 班级薄弱点
- AI 教学建议摘要

### 2. 课程资料库

模块：

- 上传区
- 资料列表
- 解析状态
- 资料摘要
- 知识点标签

### 3. 知识地图

模块：

- 知识点网络图
- 知识点详情侧栏
- 关联资料
- 关联题目
- 学生错误率

### 4. AI 出题

模块：

- 资料选择
- 知识点选择
- 题型选择
- 难度选择
- 数量设置
- 生成结果

### 5. 题库管理

模块：

- 题目列表
- 编辑题目
- 查看来源证据
- 批准/驳回
- 发布

### 6. 测验发布

模块：

- 选题组卷
- 设置测验标题
- 设置截止时间
- 发布测验

### 7. 班级分析

模块：

- 平均分
- 知识点错误率
- 能力维度分布
- 高频错题
- 风险学生
- 教学建议
