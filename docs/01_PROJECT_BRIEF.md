# 01_PROJECT_BRIEF.md

# 项目简报

## 项目名称

**BioMentor Agent｜智造学伴**

## 项目类型

AI 教育平台 / 生物制造课程智能体 / 自适应学习系统 / 科研训练平台。

## 核心用户

### 教师

- 上传课程资料
- 生成题目
- 审核题目
- 发布测验
- 查看班级分析
- 获得教学建议

### 学生

- 完成测验
- 查看解析
- 获得学习诊断
- 查看能力画像
- 进行科研案例辅导

## 竞赛表达主线

> 以科研能力训练为核心，以自适应诊断为方法，以多智能体为技术架构，以生物制造知识库为领域底座。

## 核心闭环

```text
资料 → 知识库 → 出题 → 测验 → 批改 → 诊断 → 推荐 → 教学决策
```

## 最小可演示版本

只要以下流程能跑，就算 MVP 成功：

1. 教师上传一份生物制造课程资料
2. 系统解析资料并生成摘要/知识点
3. 教师选择知识点生成题目
4. 教师审核并发布测验
5. 学生完成测验
6. 系统评分并生成解析
7. 学生看到学习报告
8. 教师看到班级分析和教学建议
