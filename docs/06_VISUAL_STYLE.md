# 06_VISUAL_STYLE.md

# 视觉风格要求

## 总体风格

高级科技感、生物制造主题、竞赛展示风格。

关键词：

```text
深蓝背景
电光蓝
青绿色
紫色 AI 光效
少量橙色强调
玻璃拟态
半透明卡片
发光数据流
知识网络
DNA 双螺旋
细胞与分子结构
生物反应器
科研论文卡片
```

## 页面风格

- 大量使用卡片式布局
- 视觉层级清晰
- 避免杂乱
- 教师端偏数据看板
- 学生端偏任务流和反馈
- 首页/展示页要更炫
- 功能页要清爽可用

## 推荐 UI 元素

- KPI 卡片
- 能力雷达图
- 知识点关系图
- 错误率热力图
- 资料状态进度条
- AI 生成结果卡片
- 教学建议面板
- 科研案例对话面板

## 主视觉提示词

```text
BioMentor Agent｜智造学伴，面向生物制造的科研型自适应学习智能体平台。16:9 高级科技海报，深蓝背景，电光蓝、青绿色、紫色 AI 光效，中心是发光 AI 智能核心，周围环绕 DNA 双螺旋、细胞、工程菌、生物反应器、分子结构、科研论文卡片、知识图谱节点和学习数据流。展示五层架构：数据层、知识层、智能体层、应用层、评价层。整体风格高级、有质感、明亮鲜艳、玻璃拟态、半透明 UI、发光线条、专业竞赛项目展示风格。
```
