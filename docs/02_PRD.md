# BioMentor Agent PRD

## 1. 产品目标

构建一个面向生物制造课程的科研型自适应学习智能体平台，支持教师上传课程资料、AI 自动出题、学生测验、智能批改、学习诊断和教师教学决策。

## 2. MVP 范围

### 教师端 P0

- 课程资料上传
- 资料解析状态展示
- AI 自动出题
- 题目审核与发布
- 测验创建
- 班级分析看板

### 学生端 P0

- 学生首页
- 测验作答
- 测验结果
- 学习诊断报告

### AI 能力 P0

- 资料摘要
- RAG 检索
- 自动出题
- 客观题评分
- 主观题评分初版
- 学习报告生成

## 3. P1 增强

- 知识地图
- 错题本
- 能力雷达图
- 科研案例辅导
- 教学建议生成

## 4. P2 延后

- 完整权限系统
- 多班级管理
- 移动 App
- 联网文献检索
- 重型知识图谱
- 考试监控

## 5. 验收标准

### MVP 验收

- 可以上传并解析资料
- 可以生成结构化题目
- 可以编辑/审核题目
- 可以发布测验
- 学生可以答题
- 系统可以评分
- 可以生成学生学习报告
- 教师可以看到班级分析

### 展示验收

- 页面风格高级、科技感、适合汇报
- 演示流程完整、不卡壳
- 有生物制造领域感
- 有五层架构表达
- 有能力画像和教学分析可视化
