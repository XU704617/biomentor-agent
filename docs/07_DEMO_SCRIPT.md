# 07_DEMO_SCRIPT.md

# 演示脚本

## 演示主题

BioMentor Agent 如何帮助生物制造课程完成从知识学习到科研能力训练的闭环。

## 演示流程

### 1. 教师上传资料

教师上传一份生物制造课程资料，例如“代谢工程与细胞工厂”。

系统展示：

- 文件解析完成
- 自动摘要
- 提取知识点

### 2. 构建知识库

系统将资料切分为知识片段，构建课程知识库。

展示：

- 知识点标签
- 资料片段
- 知识地图

### 3. AI 自动出题

教师选择“代谢工程”知识点，生成题目。

展示题型：

- 单选题
- 简答题
- 案例分析题
- 科研设计题

### 4. 教师审核题目

教师查看来源证据，编辑题目，批准发布。

### 5. 学生完成测验

学生进入测验页面，完成答题。

### 6. 系统自动评分

系统展示：

- 客观题自动判分
- 主观题 AI 评分
- 错因分析

### 7. 学生学习报告

报告展示：

- 总分
- 正确率
- 能力雷达图
- 薄弱知识点
- 推荐复习内容

### 8. 教师班级分析

教师看板展示：

- 班级平均分
- 高频错题
- 知识点错误率
- 风险学生
- 教学建议

### 9. 科研案例辅导

学生进入“工程菌株产量优化”案例。

AI 引导学生：

- 分析问题
- 提出假设
- 设计实验
- 改进方案

## 演示结论

BioMentor Agent 不只是生成题目，而是形成：

```text
资料 → 知识 → 测评 → 诊断 → 训练 → 教学改进
```

的完整智能教育闭环。
